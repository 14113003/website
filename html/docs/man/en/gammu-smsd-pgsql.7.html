NAME

gammu-smsd-pgsql - gammu-smsd(1) backend using PostgreSQL database server as a message storage

DESCRIPTION

gammu-smsd(1) supports several backends. Actual backend is chosen in a config file gammu-smsdrc(5).

PGSQL backend stores all data in a PostgreSQL database server, which parameters are defined by configuration (see gammu-smsdrc(5) for description of configuration options).

For tables description see gammu-smsd-tables(7).
