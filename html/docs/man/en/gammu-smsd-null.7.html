NAME

gammu-smsd-null - gammu-smsd(1) backend not storing messages

DESCRIPTION

gammu-smsd(1) supports several backends. Actual backend is chosen in a config file gammu-smsdrc(5).

NULL backend does not store data at all. It could be useful in case you don't want to store messages at all and you want to process then in RunOnReceive handler.
