JMÉNO

wammu-configurator - program na nastavení knihovny Gammu (kterou používá Wammu)

SYNTAXE

wammu-configure [parametry]

POPIS

Tato manuálová stránka popisuje program wammu-configure. Tento grafický program slouží k nastavení Gammu.

PARAMETRY

Tento program přijíma parametry podobně jako mnoho dalších GNU programů, kde jsou dlouhé parametry uvozeny dvěmi pomlčkami (`-'). Popis parametrů následuje dále.
-h, --help
Zobrazí přehled voleb.
-v, --version
Zobrazí verzi programu.
-l, --local-locales
Použít překlady z aktuálního adresáže místo systémových. Toto je užitečné hlavně při testování nebo při spouštění bez instalace.

LICENCE

Program je vydán pod GNU/GPL verze 2.

HLÁŠENÍ CHYB

V programu je určitě mnoho chyb a pokud je nahlásite, budeme rádi. Prosím nezapomeňte do hlášení napsat důležité informace (jako třeba jakou chybu jste dostali, případně připojit ladicí výstup). Chyby můžete zadávat na <http://bugs.wammu.eu/>.
