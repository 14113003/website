NOM

wammu - Un programme pour gérer les entrées de votre téléphone mobile

SYNOPSIS

wammu [options]

DESCRIPTION

Cette page du manuel explique le fonctionnement du programme wammu. Ce programme est une interface graphique pour gammu.

OPTIONS

Ce programme respecte la syntaxe habituelle des lignes de commande GNU avec les options longues préfixées par deux tirets ('-'). Une liste des options disponibles est incluse ci-dessous.
-h, --help
Voir la liste des options
-v, --version
Afficher la version du programme.
-l, --local-locales
Use locales from current directory rather than system ones. This is mostly useful for development or when running from unpacked tarball without installation.
-i, --info
Imprime les paramètres de connexion, tente de se connecter au téléphone et d'en afficher les informations de base. Cela n'utilise pas l'interface graphique.
-d, --debug
Active l'écriture des informations de débogage dans stderr. Fonctionne pour l'interface graphique et --info.
