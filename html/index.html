{% trans "About" %}

{% blocktrans %} Wammu is a program to manage data in your cell phone such as contacts, calendar or messages. It is built on Gammu library, which provides abstraction layer to work with different cell phones from different vendors (including Nokia, Sony-Ericssonn, Motorola, Samsung, Siemens, Huawei and others). {% endblocktrans %}

{% trans "Gammu family" %}

{% blocktrans %} Gammu is now not only a single program but there is lot of useful tools sharing same library. {% endblocktrans %}

Wammu

{% blocktrans %} GUI program to manage contacts, todos, calendar and messages in your phone. {% endblocktrans %}

Gammu

{% blocktrans %} Command line utility which allows to use all libGammu features. {% endblocktrans %}

libGammu

{% blocktrans %} Library exporting C API to talk to cell phones. {% endblocktrans %}

Gammu SMSD

{% blocktrans %} SMS daemon service to receive and send messages automatically. {% endblocktrans %}

Python-Gammu

{% blocktrans %} Python bindings for libGammu, so that you can use it from Python scripts. {% endblocktrans %}
